## Context

Our Corners, Neighborhood Stories, Founding Partners, Bottom Nav.

All CSS for each section. Static HTML (render-from-data via JS happens in T-005).
Section backgrounds must alternate per DESIGN.md spec. Bottom nav is fixed positioned.

## Acceptance Criteria

- All 8 sections present in correct render order
- Section bg alternation: hero=bg, stats+activity=bg, food=bg-warm, corners=bg, stories=bg-warm, founders=bg
- Hero: title, eyebrow, sub, live badge position, toggle placement
- Stat cards: emoji, number, label, delta — all 4 variants (color overrides per spec)
- Activity row: 2-column grid, pentagon placeholder, widget column
- Scroll rows: food, corners, stories all use .scroll-row (hidden scrollbar)
- Founders: 3-column grid
- Bottom nav: 5 tabs, fixed, frosted glass effect
