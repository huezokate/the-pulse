## Context

Build The Pulse — a mobile-first community dashboard for Mid-Market SF (Market St, 5th to Van Ness).
Hackathon prototype for (un)common ground · Frontier Tower × Human+Tech Week 2026.

Single-file HTML/CSS/JS prototype. All data hardcoded. No backend, no auth, no localStorage.

## Goal

Deliver a polished, interactive `index.html` that covers all 8 sections from the spec,
with working day/night toggle, pentagon SVG chart, bottom sheets on every tappable card,
and the full Sherwin-Williams Chinatown SF color palette.

## Acceptance

- All 8 sections render correctly on mobile (375–430px)
- Day/night toggle updates all reactive elements
- Every tappable card opens a bottom sheet
- Pentagon SVG renders from hardcoded week data, week nav arrows work
- Corner cards render all three themes (glow, plants, kids) + CTA slot
- No dark backgrounds, no Inter/Roboto, no chart libraries
- Deployable as a single file drop to Netlify/Cloudflare Pages
